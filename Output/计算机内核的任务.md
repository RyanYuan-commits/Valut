# 🌐 关键词

内核, 计算机系统

---

# 🔖 详细解释

内核是**硬件和软件**之间的**中间层**, 其作用是将应用程序的请求**传递**给硬件, 并充当底层**驱动程序**, 对系统中的各个设备和组件进行**寻址**.

- 从应用程序角度来看, 内核是一个**增强的计算机**, 将计算机抽象到了更高的层次上. 内核寻址硬盘时, 需要确定从复制数据的路径, 数据的位置等, 而应用程序只需要发送传输数据的命令, 具体如何完成与应用程序是不相干的. 内核是应用程序所知道的最底层, 因此内核是一个增强的计算机;
	
- 当若干程序在系统中并发运行时, 内核是**资源管理程序**, 将可用的资源 (如 CPU 时间, 磁盘空间, 网络连接) 等分配到各个进程;
	
- 内核是一个**库**, 提供了一组**面向系统的命令**, 通常, 系统调用用于像计算机发送请求, 借助于 C 标准库, 系统调用对于应用程序就像是调用普通函数一样.
